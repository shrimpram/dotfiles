== Installation
. git clone this repo to `~/<somedir>`, e.g. `~/dotfiles`
. cd to the dotfiles directory
. You will need https://www.gnu.org/software/stow[GNU Stow] to setup the configs
. Use `stow` to stow the modules you need. E.g. `stow nvim` will setup the neovim config

=== Installing programs
. Install https://brew.sh[Homebrew] and https://nixos.org/download.html[Nix]
. Install https://github.com/nix-community/home-manager[Home Manager]
. Stow the `nix` and `brew` directories
. Run `brew bundle --global` and `home-manager switch`
. Run `make update`

== Launchd
https://launchd.info/[`launchd`] is a daemon/process management system
specific to Mac OS. If you're familiar with
https://en.wikipedia.org/wiki/Cron[`cron`], launchd fulfills the same
main role of scheduling scripts and jobs but with added features and
utility.
