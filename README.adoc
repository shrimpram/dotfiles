== Installation
. git clone this repo to `~/<somedir>`, e.g. `~/dotfiles`
. cd to the dotfiles directory
. You will need https://www.gnu.org/software/stow[GNU Stow] to setup the configs, install it however you want
. Use `stow` to stow the modules you need. E.g. `stow nvim` will setup the neovim config

=== Installing programs
. Install https://brew.sh[Homebrew] and https://nixos.org/download.html[Nix]
. Install https://github.com/nix-community/home-manager[Home Manager]
. Stow the `nix` and `brew` directories
. Run `brew bundle --global` and `home-manager switch`
. Run `make update`

== Cron
Cron is a job scheduler I use for various repetitive tasks that I don't want to have to do every day.

=== Installation
* Cron and crontab should be installed on most *nix machines
* `cd cron` and run `crontab jobs.cron` to fill your user's crontab file with my cron jobs

=== Explanation
Below are explanations for every cron job in my file.

---

[source, cron]
----
0 * * * * mailsync
----
Syncs my mail inbox that I use with neomutt

---

[source, cron]
----
0 0 * * * make -C ~/dotfiles update
----
Runs the `update` make rule in the Makefile, which updates programs installed through both homebrew and nix.
